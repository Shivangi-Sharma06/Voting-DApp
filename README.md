# 🏦 **Decentralized Voting DApp**  

A fully **decentralized, secure, and transparent** voting system built on blockchain. No central authority—just pure trustless democracy! 🔗💙  

## 🚀 **Features**  

### **🔹 Controller (Admin) Privileges**  
✅ Add new candidates 🏆  
✅ Remove existing candidates ❌  
✅ Change the controller (admin) 🔄  

### **🔹 Voter Features**  
✅ Vote for your preferred candidate 🗳️  
✅ Pay a **minimum crypto fee** to cast your vote 💰  
✅ Transactions are **immutable & transparent** on the blockchain  

## 🔗 **Tech Stack & Tools**  

| Technology | Purpose |
|------------|---------|
| **Solidity** | Smart contract development ⚙️ |
| **Hardhat** | Local blockchain & deployment 🏧 |
| **Ether.js** | Blockchain interaction 🌐 |
| **Alchemy** | Node provider for seamless connectivity 🔑 |
| **Metamask** | Wallet integration for transactions 💳 |
| **HTML, CSS, JS** | Frontend for an interactive UI 🎨 |

## 🎨 **User Interface**  
A **clean, minimal, and user-friendly** UI that makes voting **simple & efficient** for all users! 🖥️✨  

## 📌 **How It Works**  

1️⃣ **Controller sets up candidates**  
2️⃣ **Voters connect their Metamask wallets** 🔗  
3️⃣ **Voters pay the minimum crypto fee & cast their vote** 🗳️  
4️⃣ **Votes are recorded securely on the blockchain** 🔥  
5️⃣ **Results are fully transparent & tamper-proof** 🚀  

## 🔥 **Deployment**  
- Deployed on **Hardhat**  
- Connected using **Alchemy & Ether.js**  
- **Hosted on GitHub Pages** for easy public access 🌐  

## ❤️ **Why This DApp?**  
✅ **Fully decentralized** (No single authority)  
✅ **Immutable records** (No vote manipulation)  
✅ **Secure & trustless transactions**  
✅ **Transparent & fair elections**
